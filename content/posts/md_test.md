```python
# Python 示例
def fibonacci(n: int) -> list[int]:
    a, b = 0, 1
    result = []
    for _ in range(n):
        result.append(a)
        a, b = b, a + b
    return result

print(fibonacci(10))
```

## 长段落

这是一段较长的正文，用于检测在最大宽度限制下，中文字体的行宽、字间距、行间距是否处于舒适的阅读范围。理想情况下每行约 30—35 个汉字，行高 1.8 以上，段落间距约为一个行高。同时观察标点符号是否正常显示，中英文混排时基线是否对齐 —— hello world 12345 —— 以及数字和英文单词在与汉字相邻时的间距表现。当一行文字抵达容器边界时，换行行为是否自然，是否存在孤字或标点溢出等问题，都是需要关注的重点。
